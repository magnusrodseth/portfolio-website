Revolve Aerolyze

Project information

  • Category: Full-stack web application
  • Client: Revolve NTNU
  • Project URL: aerolyze.azurewebsites.net
Revolve Aerolyze is a private product. Hence, you may not be able to access it.

A Java program that generates hundreds of images as seen in the web application will upload the images to Azure Blob Storage, and store metadata for the image in the Azure Database for PostgreSQL. Image metadata and url to a given blob image are stored in the PostgreSQL server. The Node servers makes requests to the database to find desired images. This data is then provided to frontend through the backend server supplying endpoints the frontend can make calls to. React is used as frontend framework. Redux is used for managing the state of the application.
